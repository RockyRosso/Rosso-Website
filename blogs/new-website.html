A New Website

Welcome to the new Rocky Rosso website! This website has been in development for about two months and a lot of work has gone into it. This website is still in development and will be updated occasionally. Since this site isn't being limited by a website editor, experimental features such as a chatroom, and other things that I've not thought of yet will probably be added to the site.

I would also like to mention that this website was made simply for the fun of it and no monetization such as ads, subscriptions, etc will be added to the site. The only thing that this website contains are trackers that simply measure analytics and nothing else. Analytics help me figure out how to improve the website for the best experience possible. This project has been open sourced as well which can be found here. Feel free to send me feedback via one of my socials which can be found here. Have fun exploring the site!
